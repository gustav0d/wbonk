## Getting Started


## Installation

Clone the repo

```sh
git clone https://github.com/gustav0d/wbonk.git
```

1. Install packages

   ```sh
   pnpm install
   ```

2. Run containers

   ```sh
   pnpm compose:up
   ```

3. Run the Project

   ```sh
   pnpm dev
   ```
